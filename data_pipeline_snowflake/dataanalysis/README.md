# Data Analysis Queries

This folder contains SQL queries generated by the Data Troubleshooter Agent.

## 📁 Folder Structure

- **queries/** - Ad-hoc analysis queries
- **tests/** - Data validation and quality tests
- **reports/** - Standard reports and dashboards

## 🚀 Usage

### Execute a Query
1. Right-click any `.sql` file
2. Select "Execute SQL Query"
3. View results in SQL Results Panel

### Share with Team
All queries are version-controlled via Git:
```bash
git add dataanalysis/
git commit -m "Add customer revenue analysis"
git push
```

### Use in CI/CD
Run data tests automatically:
```yaml
# .github/workflows/data-tests.yml
- run: execute-sql dataanalysis/tests/*.sql
```

## 📝 File Naming Convention

Files are named: `{date}_{purpose}_{platform}.sql`

Examples:
- `2025-01-05_top_customers_snowflake.sql`
- `2025-01-06_revenue_analysis_postgres.sql`

## 🔒 Privacy & Security

- Query results are NOT committed to Git (see .gitignore)
- Only SQL queries are version-controlled
- Queries execute locally via MCP - no data sent to LLM
